# Sales Performance Analysis - Power BI

## 📊 Project Overview
This project is a comprehensive Sales Analysis Dashboard built in Power BI. It transforms raw sales data into a three-page interactive report designed to help stakeholders understand store performance, product category distribution, and quantity trends.

## 🚀 Key Features
* **Executive Summary (Page 1):** Real-time KPI tracking using Card visuals and column charts for sales volume.
* **Category Deep-Dive (Page 2):** Hierarchical Treemaps showing product category dominance.
* **Store Distribution (Page 3):** Comparison of performance across 10+ store locations using Donut charts and clustered bar charts.

## 🛠️ Technical Implementation
* **Data Type Management:** Converted store identifiers to 'Whole Numbers' to ensure logical numerical sorting (1, 2, 3...) instead of alphabetical.
* **Axis Optimization:** Configured Store Number axes as **Categorical** to prevent Power BI from skipping labels, ensuring 100% visibility for all store locations.
* **Sorting Logic:** Implemented dual sorting capabilities allowing users to view data by either 'Store Number' (Sequence) or 'Total Quantity' (Performance).
* **Dynamic Filtering:** Integrated advanced slicers across all pages for a fully interactive user experience.

## 📈 Key Insights
* **Store Performance:** Identified the specific store locations contributing the highest 'Total Quantity' to the business.
* **Category Breakdown:** Visualized how different segments contribute to the overall sales mix, allowing for targeted inventory decisions.
